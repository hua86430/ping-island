# v0.25.5

展開島的用量條現在也顯示 Claude 額度,並修正設定滑桿拖曳與幾處中文顯示。

## 新增

- 展開島左上的用量條現在會顯示 Claude 的 5 小時 / 7 天額度(先前只有 Codex）。資料透過 Claude Code 的官方 OAuth 用量端點取得(讀取 Claude Code 存在登入鑰匙圈的憑證),並以 3 分鐘為節流窗、避免頻繁請求。

## 修改

- 用量條改版:Claude 與 Codex 各佔一行,每個時窗顯示百分比加重置倒數,時窗之間以 `|` 分隔。重置時間已經過去的時窗會自動隱藏 —— 那代表配額視窗已重置、快取的百分比是過期的舊數字,顯示反而誤導,所以整個 provider 會等到有新資料才重新出現。
- 通知 feed 的文字大小現在會跟著設定裡的「內容字級」滑桿變化(先前 feed 不受該設定影響,只有工作階段列表與滑鼠移入預覽會變)。

## 修復

- 設定視窗:按住滑桿的把手左右拖動時,不再誤拖整個視窗;要移動視窗請從上方標題列拖曳(與 macOS 系統設定一致)。
- 通知 feed 的「清除全部」按鈕在英文介面下正確顯示為 "Clear All"(先前無論介面語言都顯示中文)。
- 用量條的「剩餘 / 重置」文字修正為繁體(先前誤用簡體「剩余 / 后重置」)。
