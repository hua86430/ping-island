# v0.25.3

設定視窗大改版：換成原生 macOS 視窗外框與 NavigationSplitView 分頁側邊欄，並在 macOS 26 上採用 Liquid Glass 外觀。

## 新增

- 設定視窗在 macOS 26 上採用 Liquid Glass 外觀：側邊欄改為原生 Liquid Glass 導覽層，分類列變成扁平單行 + 彩色圖示 + 系統原生選中高亮。（在 macOS 26 上呈現；較舊系統維持原生外觀，不受影響。）

## 修改

- 設定視窗改用原生 macOS 視窗外框：標準紅綠燈（關閉／最小化／縮放）、原生視窗陰影，取代先前自刻的無邊框視窗與假的紅綠燈。
- 設定介面改為原生 NavigationSplitView：左邊分類側邊欄、右邊內容分頁，取代舊的單頁捲動版面；十個分類（通用、顯示、統計、寵物、音效、整合、遠端、實驗室、快捷鍵、關於）功能不變。
- 視窗尺寸與內容排版改用系統 safe area 自動對齊：內容自動讓開紅綠燈、自動填滿視窗，不再用固定位移；視窗維持設定的尺寸、不再被內容撐大。
- 內部重構：原本 6900 多行的設定單一大檔拆成一檔一責的模組結構（外殼、側邊欄、內容路由、各分類各一檔），只搬移不改行為。

## 修復

- 設定視窗現在可以拖曳移動（先前無邊框版本拖不動）。
- 原生的關閉與最小化按鈕都正常作用（先前右上第二顆按鈕沒有效果）。

## 移除

- 移除設定視窗早已無用的 popover 呈現模式與其相關死碼。
